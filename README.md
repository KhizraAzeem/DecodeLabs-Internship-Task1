# 🌍 Travel Explorer

Travel Explorer is a clean and responsive travel-themed landing page built using **HTML5** and **CSS3**. It showcases beautiful destinations around the world and provides a simple layout for a travel website or blog.

---

## ✨ Features

### 🧭 Header Section
- Website title and tagline
- Clean and centered design

### 📌 Navigation Bar
- Simple horizontal menu
- Hover underline effect
- Mobile-friendly stacking on small screens

### 🌄 Hero Section
- Full-width background image
- Overlay text box for readability
- Strong travel-themed headline

### 📖 About Section
- Short introduction about the platform
- Simple and readable layout

### 🏝️ Popular Destinations Section
Includes three featured travel cards:
- Mountain Escape 🏔️
- Sunny Beaches 🏖️
- Modern Cities 🏙️

Each card includes:
- Image
- Title
- Description
- Hover animation effect

---

## 📱 Responsive Design

The website is fully responsive and adapts to:

- 💻 Desktop screens
- 📱 Mobile devices

### Mobile Adjustments:
- Navigation links stack vertically
- Hero text size reduces for better readability
- Cards wrap automatically

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox Layout
- Media Queries
- Unsplash Images (external URLs)
